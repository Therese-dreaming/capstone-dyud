🔧 TROUBLESHOOTING:

Apache won't start?
→ Check if port 443 is in use: netstat -ano | findstr :443
→ Check logs: C:\xampp\apache\logs\error.log

Design/CSS not loading?
→ Clear browser cache (Ctrl+Shift+Delete)
→ Make sure .env has HTTPS URLs
→ Run: php artisan config:clear

Camera not working?
→ Must use HTTPS (not HTTP)
→ Must accept security certificate
→ Grant camera permissions when asked
